## <span style="color:#0b89ff">■ </span>연산자

- 산술 연산자
  - 기본적인 수학 계산.
  - +, -, *, /, //, %, **
```
num1 = 15
num2 = 4
print(num1 + num2)      # 덧셈: 19
print(num1 - num2)      # 뺄셈: 11
print(num1 * num2)      # 곱셈: 60
print(num1 / num2)      # 나눗셈: 3.75
print(num1 // num2)     # 몫: 3
print(num1 % num2)      # 나머지: 3
print(num1 ** num2)     # 거듭제곱: 50625
```

- 비교 연산자
  - 두 값을 비교하여 조건 판단. (true, false)
  
  - == (동등)
    ```
    x = 5
    y = 5
    print(x == y)  # 출력: True
  ```

  - != (부등)
    ```
    x = 5
    y = 3
    print(x != y)  # 출력: True
    ```

  - <, >, <=, >= (크기)
    ```
    x = 5
    y = 3
    print(x > y)   # 출력: True
    print(x <= y)  # 출력: False
    ```
  
  - 3 < a < 5 (연쇄)
    ```
    age = 20
    print(age >= 18)      # 출력: True
    print(age < 18)       # 출력: False
    print(17 < age < 23)  # 출력: True

- 논리 연산자
  - 여러 값을 비교하여 조건 판단.

  - and
  - or
  - not
  





```
# 변수
a, b, c = 1, 5, 3

# 값 변경
a = c
a = a + c
b = b - c

# 출력
print(a)
print(b)
print(c)

# 출력 : 6, 2, 3
```
